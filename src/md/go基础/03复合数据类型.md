## 数组

    数组是一个由固定长度的特定类型元素组成的序列，一个数组可以由零个或多个元素组成。
    因为数组的长度是固定的，因此在Go语言中很少直接使用数组。
    和数组对应的类型是Slice（切片），它是可以增长和收缩的动态序列，slice功能也更灵活，

    静态数组声明：
         var arr [3]int    //默认初始化每个数组元素都为0

    数组初始化：
        var arr [3]int = [3]int{1,2,3}
    
    数组的长度是根据初始化值的个数来计算：
        var arr [...]int = [...]int{1,2,3,4,5,6}

    根据索引初始化: 初始第100位为-1
        arr := [...]int{99:-1}
    
    数组的遍历：
        1.根据索引遍历：
            for i:=0;i<len(arr);i++{
                fmt.Println(arr[i])    
            }
        
        2.for range遍历 i 索引 v 值
            for i,v = range arr {
                fmt.Println(i,v)
            }

    多维数组
        var arr [2][3]int

    多维数组初始化
        arr := [2][3]int{
                    [3]int{1,2,3},
                    [3]int{4,5,6}
                }

    多维数组遍历
        for _,v = range arr{
             for _,tem = range v{
            
                }       
        }

## Slice(切片)

        和list集合相似
        
        切片指向了一个底层数组
    
        切片的长度就是它元素的个数

        切片的容量是指     底层数组从切片的第一个元素到最后一个元素的长度
            cap(arr)
            
        切片的声明
            var a1 []int
    
           切片的本质，其实际是底层声明一个数组，指向一块连续的内存，然后进行切片封装
    
        初始化
            var a1 []int{1,2,3}

    由数组得到切片：
        a1 := [...]int{1,2,3,4,5,6}
        //基于一个数组切割，左闭右开区间
        a2 := a1[:4] // -> [0:4)
        a3 := a1[4:] // -> [4:len[a1])
        a4 := a1[0:] // -> [0:len[a1])
    
    切片再切片：
        指在切片的基础之上继续切片

    使用make()函数构建切片
        make([]T,size,cap)
        make([]T,size)  //长度和容量相对
        
    切片的赋值拷贝

        s3 := []int{1,2,3}
        s4 = s3
        //更改s3的值s4跟着更改
    
    append()为切片追加元素追加元素

        s1 := []string{"asd","dfas","qwe"}
        //调用append()放大必须使用原切片变量接收
        s1 = append(s1,"asdasd")
        
        s2 := []string{"asd","dfas","qwe"}
        //...表示将s2拆开
        s1 = append(s1,s2...)
    
    copy
        copy(目标切片，源切片)

    指针
        取地址：var n = 15   &n可以去出该变量对用的地址
        根据地址取值： *n   根据指针地址取值

    make()和new
        都是用来申请内存的
        new 很少用，一般给基本数据类型申请内存，返回对应类型的指针（*int ...）
        make 用来给slice、map、chan申请内存，make函数返回的对于的这三个类型本身

    
## Map
    无序的key-value的数据结构
    
    map语法
        map[KeyType]ValueType

    进本声明
        var m = map[KeyType]ValueType
    使用make进行语法初始化,初始值位nil
        make(map[KeyType]ValueType,[cap])

    接收值方式
        value,ok := m[keyType]  
        //可根据ok进行判断，不用改判但 没有该类型的值则返回0

    delete(map,key)根据key进行删除
